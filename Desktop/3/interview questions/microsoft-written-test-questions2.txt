Microsoft Written test Questions
There were 3 questions. 10 marks each.
1. Write a program to output all elements of a binary tree while doing a
Breadth First traversal through it.



2.Write a method to combine two two sorted linked list into one in
sorted form with out using temporary Node.
void sort(Node* list1,Node* list2)



3.There are set of coins of {50,25,10,5,1} paise in a box.Write a
program to find the number of ways a 1 rupee can be created by grouping
the paise.
